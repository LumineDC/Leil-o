## Sistema de Leilões

#### Sobre o projeto:
- Este projeto consiste no desenvolvimento de um sistema informatizado para uma casa de leilões, com o objetivo de facilitar o gerenciamento de produtos orfertados nos leilões(a partir de cadastro e listagem de produtos e consulta de vendas), tornando o processo mais eficiente, seguro e organizado.

#### Tecnologias utilizadas
- Java
- MySQL
- Git e GitHub
